# Calculator
A Calculator that performs many mathematical functions

## Features
 - Add
 - Multipy
 - Divide
 - Square
 - Square Root
 - Power of n
 - Natural exponential function
 - Factorial of n
 - Trigonometric operation
 - Cuberoot
 - nPr[Permutation of n elements]
 - nCr[Number of combinations for given set]

## Tech Stack
 - HTML
 - CSS
 - JavaScript
